# Custom ls

This program is ls like with flags -a (always), -l, -h, -t, -R, --version, --help.

## Installation
```bash
make
```

## Usage

```bash
./ls.out -l [PATH]
./ls.out -l /
./ls.out -l ..
./ls.out -l /home/
./ls.out -lR /home/
./ls.out --version
./ls.out -lthR
./ls.out --version
```
